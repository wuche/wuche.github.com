最近在看《编程之美》，里面的例子蛮好玩的。中国象棋的将帅问题

思考了一下，可以这样来解决：  
遍历A的位置  
遍历B的位置  
判断A,B的位置，是否满足条件  
输出A,B的位置  
类似双层的循环 
<img src="/images/posts/c2.jpg" width="100%"  />   
要证明A,B不在同一列，只要A,B的格子点数mod3不相同。
  
    public class ChessTest {
    	public static void main(String[] args){
    		for(int i=1;i<=9;i++){
    			for(int j=1;j<=9;j++){
    				if(i%3 != j%3)
    					System.out.println("a=" + i + ",b=" + j);
    			}
    		}
    
    	}
    
    }
